Découverte du HTML et CSS

Qu'est-ce que HTML et CSS ?

HTML (HyperText Markup Language) est le langage qui structure le contenu d'une page web. CSS (Cascading Style Sheets) est le langage qui définit l'apparence et la mise en forme.

Différences entre HTML/CSS et langages de programmation

  • HTML et CSS sont des langages de balisage - ils décrivent le contenu
  • Python, JavaScript sont des langages de programmation - ils contiennent de la logique
  • HTML/CSS = Structure et apparence
  • Langages de programmation = Actions et calculs

Qu'est-ce qu'un CMS ?

Un CMS (Content Management System) est un système qui permet de créer et gérer un site web sans coder directement.

Exemples de CMS populaires

  1. WordPress - Le plus populaire, idéal pour les blogs
  2. Drupal - Plus technique, pour des sites complexes
  3. Joomla - Compromis entre simplicité et puissance
  4. Wagtail - Basé sur Python/Django

Avantages et inconvénients des CMS

Avantages :

  • Rapide à mettre en place
  • Pas besoin de coder
  • Interface visuelle intuitive
  • Grande communauté et plugins

Inconvénients :

  • Moins flexible qu'un développement sur mesure
  • Peut être plus lourd et lent
  • Dépendance aux mises à jour
  • Risques de sécurité si mal configuré

Cycle de vie d'une application web

  1. Idée - Définir le besoin et les objectifs
  2. Développement - Coder le site (HTML, CSS, JavaScript...)
  3. Test - Vérifier le fonctionnement et corriger les bugs
  4. Mise en ligne - Déployer sur un serveur d'hébergement
  5. Évolution - Maintenance et ajout de nouvelles fonctionnalités
